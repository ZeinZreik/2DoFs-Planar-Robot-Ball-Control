# 2-DOF Planar Robot Ball Control

A control and robotics project for a two-degree-of-freedom planar robot tasked with controlling a ball. The project sits at the intersection of mechanism design, dynamic control, and mechatronics experimentation.

## Project Snapshot

- Domain: robotics, control systems, mechatronics
- System: 2-DOF planar robot with ball-control behavior
- Focus: actuation, motion control, and dynamic response
- Portfolio value: demonstrates applied control work on a physical or simulated robotic mechanism

## What This Project Demonstrates

- Designing control behavior for a coupled mechanical system
- Working with two-axis motion and real-time response constraints
- Applying mechatronics thinking across mechanics, electronics, and software
- Framing a robotics problem around measurable motion and stability goals
